# Mobile-Based Secure Offline Facial Recognition and Liveness Detection System

## Features
- Offline face enrollment
- Offline face recognition
- Blink-based liveness detection
- Face encoding storage
- Mobile-friendly architecture
- SQLite database
- Security-focused design

## Requirements
Python 3.10+

pip install opencv-python face-recognition numpy sqlite3

## Run
python main.py

## Project Structure
- main.py
- face_enrollment.py
- face_recognition_module.py
- liveness_detection.py
- database.py
- requirements.txt

This is a starter implementation intended for academic/project use.
